# MPI Assignment #

#### To build just use Makefile! 
#### Below you can see list of available programs with their arguments 

🐰 **parallel_balanced** 
   * initialisation mode - *s* or *r*, sorted or random
   
🦍 **parallel_imbalanced**
   * initialisation mode - *s* or *r*, sorted or random
   
🐢 **sequential**
   * initialisation mode - *s* or *r*, sorted or random 
   * test function mode - *b* or *i*, balanced or imbalanced
